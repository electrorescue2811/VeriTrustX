# VeriTrustX

> QR-Verified Digital Governance System  
> Developed by Team E-ResQ

VeriTrustX is a secure, scalable, and real-time digital identity verification system designed for NGOs and social impact organizations.  
It eliminates identity fraud and replaces manual verification processes with encrypted QR-based authentication.

## ğŸ”„ System Workflow

1. Admin registers NGO staff
2. System generates a Unique UUID
3. Encrypted QR code is created
4. QR is embedded in:
   - Digital ID Card
   - Appointment Letter
   - Membership Certificate
5. On scan:
   - QR is decrypted
   - Hash is validated
   - Real-time status is displayed (Valid / Expired / Revoked)
